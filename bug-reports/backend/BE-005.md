# BE-005 — `PATCH /conversations/{id}/messages/read-all` has zero HTTP test coverage

**Severity:** Medium (auth/permission logic untested at HTTP layer)
**Component:** `app/routers/messaging.py:150`, `tests/`
**Status:** CLOSED — 5 HTTP-level tests added to `tests/api/test_messaging_router.py` (commit `d37a77b`). Covers 401/422/403/200/idempotent-zero cases. All 522 tests pass.

## Summary

The mark-all-read messaging endpoint exists in production but has no HTTP test verifying its auth/permission contract. Only `MessagingService.mark_all_messages_as_read` (the service method) has unit tests in `tests/unit/test_messaging_service.py:1182+`. Routing, JWT extraction, path-param parsing (UUID), and the 403 branch for non-participants are unverified end-to-end.

## Why this matters

Messaging endpoints carry strict permission rules (only conversation participants may mark messages read). The HTTP layer is where these rules become real for clients. A regression that, e.g., dropped the participant check from the router decorator would only be caught by an HTTP-level test.

## Verification of the gap

```bash
grep -lE "messages/read-all" tests/api/*.py tests/unit/*.py 2>/dev/null
# → no matches
grep -nE "read-all|mark_all_messages" tests/api/test_messaging_router.py
# → 0 results in api/, only service-level tests in unit/
```

## Suggested coverage to add

Drop into `tests/api/test_messaging_router.py`:

```python
def test_mark_all_read_non_participant_returns_403(client, override_auth, monkeypatch):
    override_auth()
    monkeypatch.setattr(
        MessagingService, "mark_all_messages_as_read",
        lambda *a, **k: (_ for _ in ()).throw(HTTPException(403, "not participant")),
    )
    r = client.patch(f"/conversations/{uuid4()}/messages/read-all")
    assert r.status_code == 403

def test_mark_all_read_idempotent_with_zero_unread(client, override_auth, monkeypatch):
    override_auth()
    monkeypatch.setattr(
        MessagingService, "mark_all_messages_as_read",
        lambda *a, **k: {"success": True, "marked_count": 0},
    )
    r = client.patch(f"/conversations/{uuid4()}/messages/read-all")
    assert r.status_code == 200
    assert r.json()["marked_count"] == 0

def test_mark_all_read_without_auth_returns_401(client):
    r = client.patch(f"/conversations/{uuid4()}/messages/read-all")
    assert r.status_code == 401

def test_mark_all_read_invalid_uuid_returns_422(client, override_auth):
    override_auth()
    r = client.patch("/conversations/not-a-uuid/messages/read-all")
    assert r.status_code == 422
```
