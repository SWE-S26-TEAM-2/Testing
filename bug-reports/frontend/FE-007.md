# FE-007 — E2E `apiRegister` missing required `display_name` field → 422 on every fresh registration

**Severity:** High (blocks auth-real + profile-real test suites from running at all)
**Component:** `e2e/real/fixtures/api-client.ts:163`, `e2e/real/auth-real.spec.ts:39`, `e2e/real/profile-real.spec.ts:29`
**Status:** FIXED — `display_name` defaulted to `username` when omitted; all callers updated (`tests/fe-qa`, commit pending).

## Summary

`apiRegister()` sent `{ email, username, password }` but the backend `RegisterRequest` schema
requires `display_name: str` (no default). Every call returned HTTP 422 instead of 201,
causing:

- `auth-real.spec.ts` "register a brand-new user" to assert `expect([200,201,409]).toContain(422)` → **FAIL**
- `profile-real.spec.ts` `resolveTargetUsername()` to return `null` (422 ≠ 409, guard hits `return null`) → follow test **always skips**

## Verification

```bash
curl -s -X POST http://localhost:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"x@x.com","username":"x","password":"X1pass!"}' | jq .
# → 422 {"detail":[{"loc":["body","display_name"],"msg":"Field required",...}]}
```

## Root cause

`RegisterRequest` (Backend `app/schemas/auth_schema.py:18`):
```python
class RegisterRequest(BaseModel):
    email: str
    username: str
    password: str
    display_name: str   # ← required, no default
```

`apiRegister` (before fix):
```ts
export async function apiRegister(payload: {
  email: string;
  username: string;
  password: string;
}): Promise<ApiResponse> {
  return apiRequest('/auth/register', { method: 'POST', body: payload, token: null });
}
```

## Fix applied

`e2e/real/fixtures/api-client.ts` — default `display_name` to `username` when not supplied:
```ts
export async function apiRegister(payload: {
  email: string;
  username: string;
  password: string;
  display_name?: string;
}): Promise<ApiResponse> {
  const body = {
    ...payload,
    display_name: payload.display_name ?? payload.username,
  };
  return apiRequest('/auth/register', { method: 'POST', body, token: null });
}
```
